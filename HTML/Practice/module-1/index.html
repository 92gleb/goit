Inline - Строчные элементы

  1. "a" ссылка
  2. "button" кнопка
  3. "img"
  4. "span" служит для выделения части информации
  5. "i" italic
  6. "b" bold делает шрифт толстым
  7. "u" underline

    Block - Блочные элементы

  1. "head" содержится техническая информация. Не отображается на странице. Входять мета теги, тег тайтл
  2. "body" Все, что отображается на странице. Входят теги header, main, footer
  3. "header" шапка сайта. Содержит логотип, навигационное меню
  4. "main" основное тело документа
  5. "footer" подвал сайта. Здесь распологаются ссылки, контакты, доп. информация. Не обязателен на сайте
  6. "nav" Навигация по сайту. Находится в header
  7. "article" статья или часть сайта. Если можно вынести эту часть в другой документ и она будет иметь смысл
  8. "section" часть сайта, которая может существовать только в контексте этой страницы, если вынести секцию за пределы страницы, то она не будет иметь смысла
  9. "div" контейнер
  10. "h1-h6" заголовки текста Н1 может быть только один на странице(его может небыть вовсе) Н2-Н6 может быть много на странице
  11. "ol" Маркированый список. Могут иметь только тег li
  12. "ul" Немаркированый список. Могут иметь только тег li
  13. "p" Параграф текста
