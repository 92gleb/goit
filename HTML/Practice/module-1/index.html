Inline - Строчные элементы

  • "a" ссылка
  • "button" кнопка
  • "img"
  • "span" служит для выделения части информации
  • "i" italic
  • "b" bold делает шрифт толстым
  • "u" underline

    Block - Блочные элементы

  • "head" содержится техническая информация. Не отображается на странице. Входять мета теги, тег тайтл
  • "body" Все, что отображается на странице. Входят теги header, main, footer
  • "header" шапка сайта. Содержит логотип, навигационное меню
  • "main" основное тело документа
  • "footer" подвал сайта. Здесь распологаются ссылки, контакты, доп. информация. Не обязателен на сайте
  • "nav" Навигация по сайту. Находится в header
  • "article" статья или часть сайта. Если можно вынести эту часть в другой документ и она будет иметь смысл
  • "section" часть сайта, которая может существовать только в контексте этой страницы, если вынести секцию за пределы страницы, то она не будет иметь смысла
  • "div" контейнер
  • "h1-h6" заголовки текста Н1 может быть только один на странице(его может небыть вовсе) Н2-Н6 может быть много на странице
  • "ol" Маркированый список. Могут иметь только тег li
  • "ul" Немаркированый список. Могут иметь только тег li
  • "p" Параграф текста
